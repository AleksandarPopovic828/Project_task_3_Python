Project: IzziHR.com
Loan Installment Calculator
Description:A system where employees enter daily tasks and receive a report on their working hours.
Project Goals:
To enable employee data entry:
Total number of employees
Total number of tasks per employee
Description of each task
Number of hours per employee for all tasks

To calculate:
Total number of hours for all employees
Total number of hours per employee
To display the results in the output in a clear and readable manner.

Пресметувач на рати за кредит
Опис:Систем каде вработените внесуваат дневни задачи и добиваат извештај за работното време.

Вкупен број на вработени
Вкупен број на задачи по вработен

Број на часови по вработен за сите задачи

Да се изврши пресметка на:
Вкупен број на часови за сите вработени
Вкупен број на часови по вработен
